Vision Based Manipulation and Grasping Using 7-DOF Robotic Arm(Jan 2020 - June 2020)

Mentor: Dr. Suril V. Shah | Done In colaboration with (IISU) ISRO Inertial Systems Unit

Technologies: Robot Operating System, Reachy arm, MoveIt, Dynamixel, Gazebo, Kinect
Keywords: Robotic Manipulation, Motion Planning, Image-based control, Visual servoing, Velocity Control





A robotic arm was simulated and controlled with ROS from scratch to carry out various tasks and implement vision-based manipulation and grasping. A Reachy robotic arm was utilized as a template for the robotic arm. This project was done in collaboration with ISRO Inertial Systems Unit as a precursor to their RESPOND project.
We, as a group of two (Dhruv Krishna and I) B.Tech undergraduate students, worked on the project and accomplished the following:

  • • Created the URDF for 7-DOF Reachy and Leachy robotic arm and simulated the arm in Gazebo by adding actuators and velocity Controllers using ROS Control package
  • • Controlled real dynamixel motors via ROS and Python with custom package made using Dynamixel SDK. Implemented velocity control of dynamixel motors in ROS[Github]
  • • Added joint actuators and wrote Position and Trajectory Controller for the robot. Added a depth camera to the arm and retrieved RGB and Depth data from it.
  • • Created a MoveIt package for Motion Planning in static envirionment with obstacles, Planned the trajectory for the Reachy arm in MoveIt! with and without obstacles
  • • Implemented eye to hand image based visual servoing in Joint Space in Gazebo for the custom robot
